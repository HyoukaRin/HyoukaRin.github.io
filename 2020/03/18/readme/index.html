写在最前面的话

发表于 2020-03-18 | 分类于 Basic

欢迎来到HyoukaRin的blog！

我并不是什么有名的人，且不比那些网红，即便是在中学，大学，计算机系的范围来看，我也是那种最默默无闻，拿了奖新闻里也不会出现名字的那种人（小鸨看了不要在意，joking），因此，即便这篇博客可以被全世界拥有网络连接的人看到，真正访问这片土地的人应该都是Rin的朋友，所谓自己人。

既然是自己人，那么这篇blog里面基本也没啥不能写的，要是哪天你们的朋友真的出名了，或者这篇blog里面的技术性教程不小心被人看到，又不小心火了，那时候，我应该会把这里面所有personal的东西统统删掉，所以我千万不要出名，诸君且看且珍惜。

如果事情进展顺利，这个blog还会与一个网站产生关联，在那里，你应该可以找到一些好玩的东西。

希望通过这个blog，你可以找到一些喜欢的东西。

Statistics

把这部分内容称为stat似乎有些冒犯，但是称为traditional machine learning似乎也并不合适，总之，这些天的实验让我深刻认识到自己的内力不足，如果我有空的话，我将系统地学习ESL或者PRML，并参加传统机器学习的比赛。

这部分内容将通过读后感和技术报告的形式展现

Review

音乐好听，动漫，电影，小说好看，哲学与社会学发人深省，还有好玩的游戏，总是在qq空间发安利总显得有些low，况且吃的人也不会太多，索性放到blog上。

我自以为没有什么创作的天赋，初中时以为自己是文艺少年，事到如今，即便是写出来的技术报告也要遭到队友对于语法的吐槽，画画唱歌弹琴就更别提了。但是，我觉得我看东西的眼光应该还算可以，如果可以成为一个不算糟糕的读者，对于作家应该也算是一种贡献吧。

Personal Thoughts

我应该不会经常更新personal thoughts的内容，如果哪天你发现我频繁更新personal thoughts的内容，那一般说明你的朋友脑子瓦特了，请稍微关心一下他，免得他继续胡言乱语。
